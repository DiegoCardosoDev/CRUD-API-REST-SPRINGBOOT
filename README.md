# API REST construida com Spring Boot, Java.


## Ferramentas
- java 17 LTS
- ORM com JPA/ Hibernate
- Spring Boot
- Maven
- H2 database

## CRUD 
- Buscar todos usuários
- Buscar um usuário pelo seu id
- Deletar usuário por id
- Atualizar usuário
- Inserir um novo usuário
